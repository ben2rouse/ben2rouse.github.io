Job List Scraper

Automating job data collection and keyword analysis with Python and Selenium

Project Overview

The Job List Scraper is a Python-based automation tool that leverages Selenium to extract job titles from Adventist Health's careers portal. It counts keyword occurrences, tracks changes over time, and helps monitor job market trends for data-related roles. This project demonstrates web scraping, automation, and data analysis skills.

Python Selenium Data Analysis Automation

Key Features

  • Automated job title extraction from a live job portal
  • Keyword-based job count and change tracking
  • Easy-to-modify for other job boards or keywords
  • Demonstrates practical use of Selenium and Python scripting
