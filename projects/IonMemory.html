Ions Study Web App

An interactive tool for memorizing ions in General Chemistry I

What is this app?

First Visit Pop-up: The app greets new users with a pop-up explaining the interactive features: clickable ions, toggling between names and symbols, and the four sections for polyatomic ions.
Interactive Table: The main view overlays ion symbols on the periodic table. Click any cell to toggle between the symbol and the ion's name.

This web app helps students study and memorize ions for General Chemistry I. It features an interactive periodic table, toggleable ion names and symbols, and practice tools for both monatomic and polyatomic ions. Pop-up instructions guide first-time users.

Try the Demo

How does it work?

Symbols View: A close-up of the table, showing all symbols for a subset of ions. This is the default state for each cell.
Names Toggled: The same snippet, but with two ions toggled to show their names instead of symbols. You can toggle any cell individually.
  • Click any ion to toggle between its symbol and name
  • Use the "Toggle symbols and names" button to switch all at once
  • Sections for monatomic and polyatomic ions

Polyatomic Ions & Practice

Polyatomic Practice: The polyatomic ions section lets you toggle between symbol and name, and use the input box to practice recalling names or symbols. Four buttons at the top let you filter by charge.
Practice in Action: Here, two ions have been toggled to show their names, and one practice box has been filled in to check your answer.
  • Practice input fields for memorization
  • Filter polyatomic ions by charge (All, 1-, 2-, 3-)
  • Instant feedback by toggling to check your answer

Getting Started (with Docker)

  1. Install Docker Desktop
  2. Clone this repository:
    git clone <your-repo-url>
    cd Ions
  3. Build and run the container:
    docker-compose up --build
  4. Open your browser and go to http://localhost:8080

Troubleshooting: If you see a "Mounts Denied" error on Mac, add the project folder to Docker Desktop's File Sharing settings.

Project Structure

  • index.php — Main web page
  • images/ — Monatomic ion images
  • AllComplex/, complex1/, complex2/, complex3/ — Polyatomic ion images
